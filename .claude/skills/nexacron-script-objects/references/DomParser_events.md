## onerror

XML 문자열 Parsing 중 오류가 생겼을 때 발생하는 이벤트입니다.

**Syntax**
```javascript
onerror(obj:nexacro.DomParser,e:nexacro.DomParseErrorEventInfo);
```

**Parameters**

| Name | Type | Description |
|------|------|-------------|
| `obj` | DomParser | Event가 발생한 Object. |
| `e` | DomParseErrorEventInfo | Event Object. |
