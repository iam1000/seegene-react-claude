## GRID

Grid 를 외부 파일로 변환할 때 사용하는 상수 속성입니다.

**Setting Syntax**
```javascript
this.ExcelExport00.addExportItem(nexacro.ExportItemTypes.GRID, this.Grid00, "Sheet1!A1");
```
