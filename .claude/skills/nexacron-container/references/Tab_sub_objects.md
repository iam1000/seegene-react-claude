| `ontouchmove` | TabpageControl 영역 내에서 터치 상태를 유지하며 움직일 때 발생하는 이벤트입니다. |
| `ontouchstart` | TabpageControl 영역 내에서 터치가 시작될 때 발생하는 이벤트입니다. |
| `onvscroll` | TabpageControl 의 수직 스크롤바를 스크롤 했을 때 발생하는 이벤트입니다. |
