# B.1웹 접근성 구현 참고 사이트

## B.1.1웹 접근성 연구소

한국지능정보사회진흥원에서 운영하는 접근성 정보, 교육 사이트입니다.

> [https://www.wah.or.kr:444/index.asp](https://www.wah.or.kr:444/index.asp)

## B.1.2ARIA Authoring Practices Guide

HTML 기본 요소 외에 다양한 기능 구현을 위해 만들어진 컴포넌트에서 어떤 식으로 ARIA를 적용하고 접근성을 향상할 수 있는지 가이드를 제공합니다.

> ARIA Authoring Practices Guide
> 
> [https://www.w3.org/WAI/ARIA/apg/](https://www.w3.org/WAI/ARIA/apg/)

한국어 번역본은 아래 링크를 참고하세요(원본 문서가 최종 공개 전에 번역된 콘텐츠라 일부 차이가 있을 수 있습니다).

> WAI-ARIA 작성 방법 1.2
> 
> [https://mulder21c.github.io/aria-practices/](https://mulder21c.github.io/aria-practices/)

## B.1.3네이버 접근성

포털 사이트 네이버에서 운영하는 접근성 안내 사이트입니다. 접근성에 대한 기본적인 개요와 네이버의 다양한 서비스에서 어떻게 접근성을 향상하기 위해 노력하고 있는지 설명하고 있습니다.

> 네이버 접근성
> 
> [https://accessibility.naver.com/](https://accessibility.naver.com/)

네이버 계열사 NTS에서 운영하는 널리는 접근성 관련 인식을 제고하고 다양한 정보를 공유하며 커뮤니티 활동을 주도하고 있습니다.

> 널리
> 
> [https://nuli.navercorp.com/](https://nuli.navercorp.com/)
