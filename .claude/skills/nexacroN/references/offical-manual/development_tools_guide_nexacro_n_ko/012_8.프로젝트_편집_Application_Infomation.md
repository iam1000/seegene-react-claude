# 8.1Application Variables

## 8.1.1Datasets

Dataset은 Data를 처리하는 표 형태의 기억장소입니다. 화면에 표시되는 컴포넌트와 Dataset이 바인딩된 경우 Data변경사항이 해당 컴포넌트에 자동으로 동기화되는 기능도 제공합니다.

Datasets 탭은 프로젝트 내에서 전역적으로 사용하는 Dataset을 추가, 삭제하거나 많은 양의 데이터를 재사용할 수 있게 하는 Import / Export 기능도 제공합니다.

![](https://docs.tobesoft.com/r/book_image/get/16d0999f8e075f35?0)

| | 속성 | 설명 |
| --- | --- | --- |
| 1 | Add a new Dataset | Dataset 추가 |
| 2 | Delete selected Dataset | Dataset 삭제 |
| 3 | Import dataset | 저장된 파일에서 Dataset을 Load |
| 4 | Export dataset | 현재 Dataset을 파일로 Save |
| 5 | Const Columns | Constant Data의 Columns 설정 |
| 6 | Edit data pathAdd const column to the datasetInsert const column to current positionDelete selected const column | Data path Editor 실행Constant Column 추가커서가 위치한 상단에 Constant Column 삽입Constant Column 삭제 |
| 7 | Columns | Dataset의 Columns 설정 |
| 8 | Edit data pathAdd column to the datasetInsert column to current positionDelete selected column | Data path Editor 실행Dataset의 Column 추가커서가 위치한 상단에 Column 삽입Dataset의 Column 삭제 |
| 9 | Rows | Dataset의 Row 설정 |
| 10 | Add row to the datasetInsert row to current positionDelete selected row | Dataset의 Row 추가커서가 위치한 상단에 Row 삽입Dataset의 Row 삭제 |

## 8.2.1MainFrame

넥사크로 앱의 기본 화면을 구성하는 요소입니다. 하위 구성 요소로 ChildFrame을 가지고 있으며 프로젝트와 함께 기본 생성됩니다.

트리 항목에서 프레임 구조를 확인할 수 있으며, 미리보기 화면을 지원합니다.

![](https://docs.tobesoft.com/r/book_image/get/0028b5cdacdc6c4f?0)

프레임 구조는 트리 화면의 툴바에서 변경할 수 있습니다.

| | 이름 | 설명 |
| --- | --- | --- |
| 1 | Add Child Frame | 선택한 프레임의 하위 레벨에 프레임을 추가 |
| 2 | Insert Frame | 선택한 프레임과 동일 레벨에 프레임을 추가 |
| 3 | Change Frame Type | 선택한 프레임의 타입을 변경 |
| 4 | Delete Frame | 선택한 프레임을 삭제 |

트리 또는 미리보기 화면에서도 컨텍스트 메뉴를 통해 편집할 수 있습니다.

![](https://docs.tobesoft.com/r/book_image/get/4054991925631e3d?0)

미리보기 화면에서는 mainframe을 제외한 나머지 프레임을 선택하고 드래그앤드롭으로 이동할 수 있습니다. 이동하고자 하는 위치로 프레임을 옮기려고 하면 이동할 수 있는 영역인지를 색상이 바뀌면서 표시합니다.

![](https://docs.tobesoft.com/r/book_image/get/e8ff40d60c6e9a2b?0)

Form 파일은 ChildFrame의 "formurl" 속성에 링크하여 넥사크로 실행 시 Form의 화면을 표시할 수 있습니다.

![](https://docs.tobesoft.com/r/book_image/get/93396b4e3c9e97e9?0)

ChildFrame은 MainFrame 하위에 올 수 있는 프레임으로 Form 정보를 Url 형태로 가지고 있어 해당 Form을 화면에 표시합니다. 기본적으로 Form이 로딩될 수 있는 기본 단위 화면을 의미합니다.
