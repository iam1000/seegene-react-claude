# 11.3변경된 기능

1. **대용량 Dataset의 메모리 성능 개선을 위해 Generate 시 \_setContents()에 해당하는 내용을 xml에서 json 형식으로 변경했습니다. (RP 101145)**
 

| 설명 |
| --- | --- |
| 21.0.0.1900 Tool&SDK를 사용하고 ReGenerate를 필요로 합니다.21.0.0.1900 Tool만 설치하고 SDK를 이전 버전을 사용 시 적용되지 않습니다. | |
| 변경 전 | Dataset._setContents() 함수의 내용이 xml 형태의 String 형태로 변경 |
| 변경 후 | Dataset._setContents() 함수의 내용이 Json Object 형태로 변경 |

2. **WebView 컴포넌트의 User Data 경로를 프로젝트(key) 별로 다르게 사용하도록 기능을 변경했습니다. (RP 101099)**
 

| 설명 |
| --- | --- |
| 변경 전 | AppData\Roaming\Microsoft\nexacro.WebView\User Data |
| 변경 후 | AppData\LocalLow\TOBESOFT\Nexacro N\21\WebView\{key-name} |

3. **Nexacro Studio에서 화면 소스의 개행 문자(line ending)가 LF/CRLF가 혼용되는 문제를 CRLF로 사용하도록 Studio 소스 편집기를 변경했습니다. (RP 97569)**
 

| 설명 |
| --- | --- |
| 변경 전 | LF/CRLF가 혼용 저장. |
| 변경 후 | 신규프로젝트 -> 생성되는 모든 파일이 <CRLF>로 생성.기존프로젝트 -> 파일 오픈 후 수정, 다시 저장해야 <CRLF>로 변경되어 저장. |
