# 24.1QuickCode Tutorial Wizard 실행하기

프로젝트 생성 시 \[Use 'QuickCode' > Start Tutorial\]를 선택하거나 프로젝트를 연 상태에서 메뉴\[QuickCode > Let's Try QuickCode\] 선택 시 QuickCode Tutorial Wizard를 실행합니다.

## 24.1.1프로젝트 생성 마법사에서 Use 'QuickCode' 선택하기

QuickCode 단계에서 \[Use 'QuickCode' > Start Tutorial\] 항목을 선택하고 \[Finish\] 버튼을 클릭하면 QuickCode Tutorial Wizard를 실행합니다.

QuickCode Tutorial Wizard 실행 중 \[Cancel\] 버튼을 클릭하면 QuickCode 설정 없이 프로젝트를 생성합니다.

![](https://docs.tobesoft.com/r/book_image/get/62ec2428513af071?0)

## 24.1.2메뉴에서 Let's Try QuickCode 항목 선택하기

메뉴\[QuickCode > Let's Try QuickCode\] 선택 시 QuickCode Tutorial Wizard를 실행합니다. 현재 열려있는 프로젝트에 개발 환경을 설정하고 샘플 파일을 설치합니다.

# 24.2QuickCode Tutorial Wizard 실행 화면

QuickCode Tutorial Wizard는 설정값을 입력하는 화면은 없으며 간단한 기능 소개 화면으로만 구성되어 있습니다. \[Next\] 또는 \[Prev\] 버튼을 클릭해 내용을 탐색할 수 있습니다.

![](https://docs.tobesoft.com/r/book_image/get/f630d9897bc6253f?0)

| | 설명 |
| --- | --- |
| 1 | 기능 소개 내용을 확인하고 마지막 소개 단계에서 [Finish] 버튼이 활성화됩니다.[Finish] 버튼을 클릭하면 개발 환경을 설정하고 샘플 파일을 설치합니다. |
| 2 | 실행 중 [Cancel] 버튼을 클릭하면 Wizard 실행을 종료합니다. |

# 24.3QuickCode Tutorial Wizard 실행 결과

QuickCode Tutorial Wizard를 실행하고 \[Finish\] 버튼을 클릭하면 프로젝트에 아래와 같은 결과가 반영됩니다.

- View 컴포넌트를 사용할 수 있도록 설정하고 툴바에 표시합니다.
 
- QuickView 툴바를 표시합니다.
 
- model 타입의 서비스를 등록하고 Tutorial 실행을 위한 모델 샘플을 등록합니다.
 
- Tutorial 실행을 위한 View Template을 등록합니다.
 
- Tutorial 실행을 위한 Action 오브젝트를 설치하고 등록합니다.
 
- Tutorial 실행을 위한 Viewset Template을 등록합니다.
 
- Tutorial 실행을 위한 XCSS 파일을 추가합니다.
 
- 그리고 sample\_SingleDetail Viewset Template으로 QuickCodeTutorialForm.xfdl 파일을 생성합니다.

## 24.3.1QuickCodeTutorialForm.xfdl

QuickCode Tutorial Wizard 실행 결과로 생성되는 파일이며 QuickView 실행 후 \[Search\] 버튼 클릭 시 데이터를 조회할 수 있습니다.

![](https://docs.tobesoft.com/r/book_image/get/08ab0a259df6392d?0)
