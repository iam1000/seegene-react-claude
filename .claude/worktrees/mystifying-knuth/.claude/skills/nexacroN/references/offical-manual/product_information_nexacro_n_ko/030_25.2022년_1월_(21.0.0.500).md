1. **기본 앱으로 지정한 웹브라우저를 항상 새창으로 여는 기능을 추가했습니다. (RP 93351, 93572)**
 

| 설명 |
| --- |
| 기본 앱으로 지정한 웹브라우저를 항상 새창으로 여는 기능으로 system.execDefaultBrowser() 에 bNewWindow 파라미터를 추가했습니다.bNewWindow 파라미터는 윈도우 운영체제 NRE에서만 지원합니다system.execDefaultBrowser( strUrl, [bNewWindow] )strUrl : 웹브라우저 URL 경로를 문자로 설정bNewWindow : true/false(default) 새창으로 띄울 건지 여부 |
| 자세한 내용은 "Nexacro N Reference Guide" 를 확인하시기 바랍니다.Misc. Objects > System > Method > execDefaultBrowser |
