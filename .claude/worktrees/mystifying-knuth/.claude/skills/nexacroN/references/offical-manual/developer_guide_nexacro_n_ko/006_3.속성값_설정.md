# 3.1선택한 컴포넌트를 속성창에 표기하기

디자인 화면에 배치된 컴포넌트를 선택하면 속성창에 선택한 컴포넌트의 속성값이 표시됩니다. 속성창에서는 속성값을 확인하거나 수정할 수 있습니다.

![](https://docs.tobesoft.com/r/book_image/get/5d3ae873377553d8?0)

디자인 화면에서 컴포넌트를 선택하지 않더라도 \[Project Explorer\]에서 해당 컴포넌트를 선택하거나 속성창 목록에서 원하는 컴포넌트를 선택할 수 있습니다.

![](https://docs.tobesoft.com/r/book_image/get/6356873ad59fd55e?0)

![](https://docs.tobesoft.com/r/book_image/get/7a0f349e335e1671?0)

# 3.2선택한 컴포넌트의 기본값 확인하기

컴포넌트를 디자인 화면에 처음 배치하면 기본값으로 속성값이 설정됩니다. 컴포넌트를 선택한 상태에서 넥사크로 스튜디오 오른쪽에 표시되는 속성창에서 컴포넌트의 속성값을 확인할 수 있습니다.

컴포넌트의 기본값을 수정한 경우에는 굵은 글꼴로 속성값이 표시됩니다. 디자인 화면에 처음 배치된 상태에서 id, text, left, top, width, height 속성값이 굵은 글꼴로 표시되는 것은 컴포넌트를 배치하는 과정에서 id, text 속성값이 지정되고 Position 관련 속성값이 배치된 위치에 따라 지정되기 때문에 굵은 글꼴로 속성값이 표시됩니다.

![](https://docs.tobesoft.com/r/book_image/get/2caf37c22caf7d1e?0)

초기 상태에서 enable 속성값을 false로 수정하면 아래와 같이 굵은 글꼴로 표시됩니다.

![](https://docs.tobesoft.com/r/book_image/get/96b392d7598942b5?0)

# 3.3속성창에서 Button 컴포넌트의 text 속성값 변경하기

1. 디자인 화면에 Button 컴포넌트를 배치하고 배치한 Button 컴포넌트를 선택합니다.

2. 속성창에서 text 속성 항목을 찾습니다. 속성창 상단 검색창에 원하는 항목을 입력하면 바로 찾을 수 있습니다.

![](https://docs.tobesoft.com/r/book_image/get/b672a8f839b17711?0)

속성창 상단 검색창은 검색어로 입력된 항목을 포함하는 속성 항목을 찾습니다. 검색어로 "text"를 입력하면 속성명에 "text"가 포함된 모든 속성 항목을 보여줍니다.

3. text 속성 항목에서 속성값이 있는 부분을 마우스로 클릭하고 원하는 텍스트를 입력합니다.

![](https://docs.tobesoft.com/r/book_image/get/49d5ff660dde5780?0)

텍스트 입력 항목에 2줄 이상 텍스트를 입력하고자 할때는 오른쪽에 있는 버튼을 클릭하면 2줄 이상의 텍스트를 입력할 수 있는 에디터가 표시됩니다. 줄바꿈은 Ctrl+Enter 키를 입력하면 바뀝니다.

![](https://docs.tobesoft.com/r/book_image/get/3a4e5c7b921586c1?0)

4. 텍스트 입력 후 다른 속성 항목으로 이동하거나 디자인 화면으로 포커스를 옮기면 Button 컴포넌트의 텍스트가 변경된 것을 확인할 수 있습니다.

![](https://docs.tobesoft.com/r/book_image/get/699c4f3831369ff2?0)

# 3.4스크립트에서 Button 컴포넌트의 text 속성값 변경하기

1. 스크립트 화면으로 변경합니다. 디자인 화면 하단에 보이는 3개의 탭 버튼 중에서 "Script" 탭을 선택합니다.

![](https://docs.tobesoft.com/r/book_image/get/a0e4aeb84c3c0b5c?0)

2. 스크립트 화면에서 코드를 입력합니다. 넥사크로 스튜디오 스크립트 화면에서는 코드 작성 시 도움을 주는 인텔리센스 기능을 지원합니다.

"this"라는 텍스트 입력 후 "."을 입력하면 Form 오브젝트의 속성, 메소드, 이벤트와 함께 배치된 컴포넌트의 목록을 보여줍니다. "." 입력 후 입력되는 글자에 따라 표시되는 목록을 필터링해서 보여줍니다. 목록에서 원하는 컴포넌트를 선택합니다.

![](https://docs.tobesoft.com/r/book_image/get/92bb2c47debe2e0d?0)

3. "this."을 입력한 것과 마찬가지로 "this.Button00."을 입력하면 Button 컴포넌트의 속성, 메소드, 이벤트 목록을 보여줍니다. 해당 목록 중에서 text 속성 항목을 선택합니다.

![](https://docs.tobesoft.com/r/book_image/get/bbf749eda77bd4d8?0)

4. "this.Button00.text"까지 입력한 상태에서 등호("=")를 입력하면 속성값을 설정할 수 있는 코드가 자동으로 작성됩니다.

![](https://docs.tobesoft.com/r/book_image/get/a187e56d193593d4?0)

이전 단계에서 "this.Button00." 다음에 "set\_"으로 시작하는 텍스트를 입력해도 같은 코드를 작성할 수 있습니다.

![](https://docs.tobesoft.com/r/book_image/get/85f5b45b9b1d48d0?0)

5. text 속성값으로 지정할 값을 입력합니다. 완성된 코드는 아래와 같습니다.

this.Button00.set\_text("TEST");

6. 변경된 내용을 저장하고 툴바에서 \[QuickView\] 항목을 선택합니다. 넥사크로 브라우저가 실행되면서 화면에 Button 컴포넌트가 표시됩니다. 화면을 로딩하면서 앞에서 작성한 코드를 실행하고 Button 컴포넌트의 text 속성이 바뀐 상태에서 표시되는 것을 확인할 수 있습니다.

![](https://docs.tobesoft.com/r/book_image/get/f3cce92902a38fa3?0)
