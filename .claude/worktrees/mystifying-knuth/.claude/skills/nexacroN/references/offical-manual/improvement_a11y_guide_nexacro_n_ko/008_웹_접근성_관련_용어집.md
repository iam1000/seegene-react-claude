# 웹 접근성 관련 용어집

## [Download PDF](https://docs.tobesoft.com/r/pdf/download_chapter/a41363937522a407/596cf126a44ae630)

KWCAG

KWCAG는 한국형 웹 콘텐츠 접근성 지침의 약자입니다. WCAG(Web Content accessibility guidelines)를 한국 상황에 맞게 작성한 지침입니다. 2005년 1.0 버전이 만들어졌고 2010년 2.0, 2015년 2.1, 2022년 2.2 버전이 발표됐습니다.

KWCAG는 웹 접근성을 평가할 수 있는 항목별 기준을 제시합니다. 다음 링크에서 KWCAG 2.2 문서를 내려받을 수 있습니다.

https://www.wah.or.kr:444/Participation/guide.asp

가상커서

가상커서는 센스리더의 기본 동작이며 특정 프로그램 실행 시 활성화되는 기능입니다. 화면에 표시된 웹페이지를 하나의 문서처럼 재구성해서 상하방향키로 탐색할 수 있게 재구성합니다. 웹브라우저에서 넥사크로 N에서 개발한 화면이 실행되는 경우에는 가상커서가 활성화된 상태로 동작합니다.

\* 넥사크로 N 21.0.0.1200 미만 버전에서는 가상커서의 동작을 정상적으로 지원하지 못해 사이트 이용 시 가상커서를 비활성화하고 사용하는 것을 권장했습니다. 21.0.0.1200 이후 버전에서는 이런 제약이 해소되었으며 사용자가 센스리더 기본 설정을 변경하지 않고 넥사크로 N에서 개발한 사이트를 이용할 수 있습니다.

선택상자

Static 컴포넌트처럼 포커스를 가지지 않는 컴포넌트 정보를 스크린 리더에서 읽을 수 있도록 제품 내에서 제공하는 기능입니다. 센스리더에서 가상커서를 해제한 경우에 사용할 수 있습니다.

상하방향키 입력 시 센스리더에서 가상커서를 활성화한 경우에는 센스리더 동작이 우선 적용되기 때문에 선택상자는 동작하지 않으며 가상커서 기준으로 동작합니다.

스크린 리더

화면에 표시되는 정보를 음성으로 출력하는 소프트웨어입니다. 운영체제에서 기본 기능으로 제공되기도 하고 별도 소프트웨어를 설치해 사용하기도 합니다. 모바일 기기는 각 운영체제에서 기본 기능으로 제공되는 소프트웨어를 주로 사용하며 한국의 경우 윈도우 운영체제에서는 엑스비전 테크놀로지에서 개발한 스크린 리더 소프트웨어인 센스리더를 대부분 시각 장애인이 사용합니다.

\* 센스리더는 유료 소프트웨어이지만 정부의 정보통신 보조기기 지원으로 선택할 수 있습니다. 하지만 개발자가 직접 이를 구입해 체험해 보기에는 어려움이 있어 가이드 문서 내에서는 동작 방식을 글이나 그림 형태로 설명하거나 무료로 사용할 수 있는 스크린 리더를 대체로 경험하도록 안내합니다.

접근성

접근성은 웹 접근성의 상위 개념이며 제품, 서비스, 건축 등 다양한 분야에 해당하는 용어입니다. 문서 또는 제품 도움말 내에서 넥사크로 N에서 개발한 결과물을 사용하는 경우를 이야기하며 "접근성"이라고 표현할 때 이는 "웹 접근성"과 같은 범위를 언급하는 것입니다.

접근성 영어 표기인 accessibility를 줄여서 a11y라고 사용합니다.
